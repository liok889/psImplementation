Portilla–Simoncelli Texture Model — JavaScript

A browser port of the steerable-pyramid texture analysis/synthesis algorithm (Portilla & Simoncelli, 2000), faithful to the C++ reference. Upload an image to see the pyramid decomposition and the extracted joint statistics, then synthesize a new texture with the same statistics. Grayscale path.

2 · Pyramid filters frequency-domain masks (DC at center)

High-pass H₀, low-pass L₀, and the K oriented band-pass filters at the finest scale.

3 · Pyramid decomposition filter outputs at each stage

Each row is a scale (finest at top). Cells show the magnitude |·| of each oriented sub-band, then that scale’s low-pass residual. Brightness is per-tile normalized.

4 · Extracted statistics the texture’s parametric summary

These constraints (marginal moments, raw auto-correlations, magnitude & phase cross-correlations across orientation and scale) are exactly the quantities validated against the C++ reference — see test/run_validation.js.
